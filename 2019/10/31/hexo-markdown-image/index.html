hexo markdown中引用图片

安装hexo-asset-image

配置根目录_config.yml文件中的post_asset_folder为true

hexo执行新建文件命令（hexo n xxx）后，会生成xxx.md与xxx文件夹，把图片放进xxx文件夹中，图片引入方式如下：

注：

⑴、本地图片需要上传github后，才会生效，因为最终生成图片路径是：根目录_config.yml文件配置的url + 年月日 + xxx文件 + 图片名称。

⑵、npm install hexo-asset-image –save，如直接安装可能会由于版本兼容问题，导致图片路径错误。
